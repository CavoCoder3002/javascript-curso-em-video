Aprendendo a usar o DOM em JavaScript

O que é o DOM?

-Dom é um acrônimo de "Document Object Model"

-Conjunto de Objetos dentro do navegador que dá acesso aos componentes dentro do site.

Clique em mim

Árvore DOM

~~~~window~~~~

location ; document ; history

*location: Url do site (Localização do site), página atual, página anterior

*document: Documento atual
 -|-Dentro de "document": ~html
 -|-|-Dentro de html: ~head; ~body
 -|-|-|-Dentro de head: ~meta; ~title
 -|-|-|-Dentro de body: ~h1; ~h2; ~p; ~p1; ~p2; ~p3; ~p4; ~div
*history: De onde você veio, ou onde você vai
